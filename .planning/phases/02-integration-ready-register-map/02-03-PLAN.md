## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Modbus client -> SG-ready register (5000) | Untrusted mode write from evcc crosses into physical heat pump control |
| Proxy -> Luxtronik controller | SG-ready mode translation results in real parameter writes to hardware |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-07 | Tampering | SG-ready virtual register write | mitigate | Value validated against allowed_values [0,1,2,3] in RegisterMap. enable_writes gate must be True. Invalid values rejected with ExcCodes.ILLEGAL_VALUE before reaching the controller. |
| T-02-08 | Tampering | Custom sg_ready_mode_map in config | mitigate | Config validated at startup by pydantic. Mode map values are Luxtronik parameter indices (3, 4) which are themselves validated by the existing write validation chain before reaching the controller. |
