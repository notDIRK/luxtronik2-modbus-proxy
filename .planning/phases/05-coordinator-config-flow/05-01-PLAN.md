| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-01 | Tampering | coordinator._sync_read | accept | Data comes from local LAN heat pump controller over proprietary binary protocol; no authentication exists in protocol (design constraint). Values are raw integers stored as-is. |
| T-05-02 | Denial of Service | coordinator._async_update_data | mitigate | asyncio.Lock prevents concurrent connections; UpdateFailed on exception triggers HA retry backoff instead of tight reconnect loop |
| T-05-03 | Information Disclosure | coordinator.data | accept | Data contains only heat pump operational values (temperatures, modes); no PII or credentials. Stored in hass.data which is HA-internal. |
